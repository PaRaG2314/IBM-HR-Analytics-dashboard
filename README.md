# 👥 IBM HR Analytics Dashboard

## Overview

This project presents an interactive Tableau dashboard built using the IBM HR Analytics dataset to analyze employee attrition, workforce demographics, job satisfaction, compensation patterns, and employee performance metrics.

The dashboard helps HR teams and business leaders identify key factors influencing employee attrition and workforce retention, enabling data-driven talent management decisions.

---

## Business Objective

Employee attrition can significantly impact organizational productivity and hiring costs. The goal of this project is to:

* Analyze employee attrition trends
* Understand workforce demographics
* Identify high-risk attrition groups
* Evaluate job satisfaction and work-life balance factors
* Support strategic HR decision-making through visual analytics

The dashboard is built using the IBM HR Analytics Employee Attrition Dataset containing information about:

* Job Role
* Education
* Monthly Income
* Years at Company

* Job Satisfaction
* Work-Life Balance
